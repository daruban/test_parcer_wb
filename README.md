## Парсер для карточек товаров на ВБ по API
<br>Настройка осуществляется через переменные в начале файла main.
<br>Скрипт актуален до 60-ого шарда серверов wildberries.
<br>Проект запукается через main.
<br>По завершению скрипта создаются два файла full_catalog.xlsx -- полный список, filtered_catalog.xlsx -- отфильтрованный по ТЗ список.
